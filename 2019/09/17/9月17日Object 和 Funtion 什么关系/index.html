js 原型的问题 Object 和 Function 到底是什么关系？

发表于 2019-09-17 | 更新于: 2019-09-17 | 分类于 前端 |
字数统计: 165 | 阅读时长 ≈ 1
  1. 首先Object和Function都是构造函数，而所有的构造函数的都是Function的实例对象. 因此Object是Function的实例对象
  2. Function.prototype是Object的实例对象
  3. 实例对象的原型(我们以proto来表示)会指向其构造函数的prototype属性, 因此 Object.proto === Function.prototype,Function.proto===Function.prototype,Function.prototype.proto === Object.prototype
  4. 当我们访问一个属性值的时候, 它会沿着原型链向上查找, 直到找到或者到Object.prototype.proto(为null)截止.
